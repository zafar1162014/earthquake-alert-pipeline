# Earthquake Alert Pipeline

A Pakistan-focused earthquake disaster alert pipeline using Hadoop and Apache Spark.

## Project Structure

- `scripts/01_download_data.py` - Download global + Pakistan earthquake data from USGS
- `scripts/02_upload_hdfs.sh` - Upload local CSV into HDFS
- `scripts/03_batch_analysis.py` - Batch analytics (magnitude, region, places, Pakistan stats)
- `scripts/04_hotspot.py` - Global and Pakistan hotspot detection with risk levels
- `scripts/05_stream_feed.py` - Socket-based live stream feed simulator
- `scripts/06_stream_alert.py` - Structured Streaming alert classification and sink
- `scripts/07_amdahl.py` - Amdahl's law benchmark and speedup plotting

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Typical Run Order

1. `python scripts/01_download_data.py`
2. `bash scripts/02_upload_hdfs.sh`
3. `spark-submit scripts/03_batch_analysis.py`
4. `spark-submit scripts/04_hotspot.py`
5. Terminal 1: `python scripts/05_stream_feed.py`
6. Terminal 2: `spark-submit scripts/06_stream_alert.py`
7. `python scripts/07_amdahl.py`

## Outputs

- `data/earthquakes.csv`
- HDFS outputs under `/earthquake/output/...`
- `output/speedup_chart.png` (generated by Task 6)
